Để biên dịch chương trình: g++ -g main.cpp SACA-K.cpp -o main.exe
Chạy chương trình qua file main.exe
File huffman.c là file mã nguồn của thuật toán huffman, biên dịch thu được file huffman.exe
Để chạy file huffman.exe, thực hiện câu lệnh như sau từ terminal:
	Để mã hóa: huffman encode <tên file đàu vào> <tên file đầu ra>
	Để giải mã: huffman decode <tên file đầu vào> <tên file đầu ra>


File redirect.txt là một file rỗng được sinh ra khi thực hiện điều hướng stdout từ file đến màn hình terminal.
Các file kết quả:
	File output.txt lưu trữ kết quả sau khi thực hiện biến đổi BWT-MTF.
	File result.txt lưu trữ kết quả sau khi giải mã ngược file output.txt
	File first_index.txt lưu trữ chỉ số first của biến đổi BWT
